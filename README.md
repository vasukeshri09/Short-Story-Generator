# Short Story Generator 🚀

This project is an AI-based story generation system using GPT-2 and LoRA fine-tuning.

## Features
- Generate stories from prompts
- Genre & tone control
- Story length slider
- Export to PDF
- Standalone EXE support

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Download everything:
   [https://drive.google.com/drive/folders/1_sjryospe0EmYhQNs5Jfx5y3AW77UJYC?usp=drive_link]

3. Run:
python app_v5.py
